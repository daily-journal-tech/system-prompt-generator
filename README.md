# System Prompt Generator
Generate the best system prompts based on the best system prompts in the world

Meta analysis on (https://github.com/x1xhlol/system-prompts-and-models-of-ai-tools/tree/main)

Extract the best practices, entities, insights, patterns and more, that makes a good system prompt

Tool to create yours based on your use case, with all the knowledge from the best prompts.

### How to use it

Add your Open Router API key, type your use case and the AI will generate the prompt.
